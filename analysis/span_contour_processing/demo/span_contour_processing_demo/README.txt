span_contour_processing_demo

This is the data to use for the demo of the span_contour_processing repository. It contains tracking outputs from the repository span_segmentation and real-time magnetic resonance imaging videos as avi files. 

Usage:

Download the zip archive span_contour_processing_demo and unzip. Create the folder demo_files inside the cloned span_contour_processing repository. Move the contents of the zip archive span_contour_processing_demo into the folder demo_files inside the cloned repository.
